# Daily Tagalog

A self-contained Tagalog learning MVP built with plain HTML, CSS and JavaScript.

## Run it

Open `index.html` in any modern browser.

For the most reliable local setup, run a tiny local server from this folder:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Included

- Responsive dashboard
- Daily lesson flow
- Interactive vocabulary flashcards
- Sentence translation exercises
- Grammar mini-lesson and quiz
- Short reading passage with revealable translations
- Verb trainer
- Review mode
- Progress saved with browser localStorage
- Browser text-to-speech when supported
